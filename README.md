# ISynspec

A modern Python interface for the SYNSPEC spectral synthesis program.

## Installation

For development:

```bash
pip install -e ".[dev]"
```

## Development

This project uses:
- `pytest` for testing
- `black` for code formatting
- `isort` for import sorting
- `flake8` for linting

### Running Tests

```bash
pytest
```

### Code Formatting

```bash
black .
isort .
flake8 .
```
